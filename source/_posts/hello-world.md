Welcome to [Hexo](https://hexo.io/)! 这是基于 Hexo + Butterfly 魔改的现代化个人博客。

## 快速开始

### 创建新文章

```bash
$ hexo new "My New Post"
```

### 运行服务器

```bash
$ hexo server
```

### 生成静态文件

```bash
$ hexo generate
```

### 部署到远程站点

```bash
$ hexo deploy
```

## 功能特性

- ⚡ **极速加载**: Cloudflare CDN + 静态资源优化
- 🎨 **魔改主题**: 基于 Butterfly 深度定制
- 💬 **Waline 评论**: Cloudflare Workers + D1
- 📊 **数据分析**: Cloudflare Web Analytics
- 🖼️ **R2 图床**: 自动格式转换与缓存
- 🔍 **SEO 优化**: sitemap + 结构化数据
- 🌐 **多端响应**: PWA + 移动端适配
