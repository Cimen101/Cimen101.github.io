# Cimen101's Blog

Personal blog built with Hexo and deployed on GitHub Pages.

Live: https://Cimen101.github.io

## Tech Stack

- Hexo - Static site generator
- Butterfly - Theme
- GitHub Pages - Deployment
- Cloudflare Pages - Admin panel
- GitHub Actions - Auto deployment

## Local Development

```bash
npm install
npx hexo clean
npx hexo generate
npx hexo server
```

Visit http://localhost:4000
